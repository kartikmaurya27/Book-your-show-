# Book-your-show-

🎩 Magician Parth – Event Entertainment Website
A modern and visually immersive event entertainment website built for Magician Parth, designed to showcase magical performances, event services, testimonials, and booking information with a premium user experience.

✨ Project Overview
This project is a responsive multi-page entertainment website created using HTML, CSS, and JavaScript.
The website highlights different magic show services including:


Birthday Magic Shows


Corporate Event Entertainment


Wedding Magic


Kids Party Shows


The design focuses on creating a luxurious and magical atmosphere using animated backgrounds, glowing typography, floating effects, and elegant layouts.
